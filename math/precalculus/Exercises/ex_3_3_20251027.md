$\frac{3x^3}{x}=3x^2$

$3x^2(x+5)=3x^3+15x^2$


$3x^3+5x^2+5-3x^3-15x^2=-10x^2+5$


current $r(x)=-10x^2+5$


$\frac{10x^2}{x}=10x$


current $q(x)=3x^2+10x$


$10x(x+5)=10x^2+50x$


$-10x^2+5-10x^2-50x=-50x+5$


current $r(x)=-50x+5$


$-\frac{50x}{x}=-50$


current $q(x)=3x^2+10x-50$


$-50(x+5)=-50x-250$


$-50x+5+50x+250=255$


final $r(x)=255$
