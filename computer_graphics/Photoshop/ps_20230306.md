# 강의내용
- 인쇄 해상도는 `300`, 영상 해상도는 `72`
- `Color Mode > RGB Color + 8 bit`
- `Advanced Options > Pixel Aspect Ratio > Square Pixels` 로 지정
- 이미지 파일 `Drag & Drop > Enter` 로 이미지 삽입
- 작업창에 이미지 삽입한 경우, 짧은면 기준으로 압축되어 삽입됨
- `F7` : 레이어 패널 열기/닫기
- `Smart Object` : 레이어 패널에 아이콘이 있는 객체, 원본을 가지고 있는 객체
- `Smart Object` 더블클릭 시 원본 이미지 파일 확인 가능
- `Ctrl + T` : 클릭한 객체에 대하여 `Free Transform` 을 활성화함, `Enter` 로 비활성화
- `Ctrl + J` : 레이어 복사
- `Layer Panel > Layer (Title) > R.Clck > Rasterize Layer` : 스마트이미지를 이미지화함. 작업창에 최적화된 이미지로 변환됨으로, 더 이상 `Smart Object` 가 아님.
- 작업창을 일러에서는 `Artboard` 라 불렀되, 포샵에서는 `Canvas` 라고 부름
- `Image > Image Size` | `Ctrl + Alt + I` : 이미지 사이즈 자체를 줄임
- 이미지 저장 시 `Save As` 옵션을 사용하며, `.psd` 포맷으로 저장
- `Image > Canvas Size` | `Ctrl + Alt + J` : 캔버스 사이즈를 조절함. 캔버스 이내 이미지 사이즈와 무관하되, 이미지 일부가 잘리긴 함 (Clipping)
- `300 ppi` 해상도를 가진 이미지를 가져온 경우, 해상도를 `72` 로 조정한 후 이미지 크기를 조절할 것. 순서가 중요함!!
- `좌측메뉴 > 색상 조절` : Foreground, Background 색상 조절
- `Ctrl + D` : 선택된 레이어를 기본 (Default) 색상으로 조절
- 이미지 `.psd` 파일로 작업하는 경우, 이미지 자체가 `background` 레이어로 인식됨
- `Ctrl + G` : 선택한 레이어에 대해 `Group` 폴더를 생성함

## 이미지 보정
- `Clone Stamp tool` | `S` : "도장툴"은 선택한 부분을 복사하여 다른 부분 위에 그리는 기능
- `S + [ or ]` : Clone Stamp 커서의 크기를 조절
- `F5` : 브러쉬 패널
- `S + Alt + L.Clck` : 얼트클릭한 위치의 부분을 복사함. 이후 일반 `L.Clck` 을 한 부분 위로 덛입혀짐
- "개새" 디자인 트렌드는 도장 도구 연습/대회로 부터 시작됨
- 도장 도구 사용시, 수정하려는 부분과 최대한 근접한 부분으로 선택하고 커서 크기를 작게하는게 좋음. 그리고, 수정할 부분들마다 복사 부분을 다르게 해줘야 더 자연스러워 보임
- `Healing Brush tool` | `J` : 도장툴과 유사하되, 그려지는 부분의 주변과 합쳐지는 효과가 있음
- 도장 vs 힐링
  ![[Pasted image 20230306121338.png]]
- 머리카락 처럼 대비가 크고 선명한 부분은 도장툴 권장, 스킨 처럼 부드러운 부분은 힐링툴 권장
- `Lasso tool` | `L` : 이미지 위에 선택할 부분을 그리는 도구
- `L + J` : 라쏘로 선택한 영역 부분 내에서만 수정이 가능토록 해줌
  ![[Pasted image 20230306122457.png]]
- `Ctrl + D` : 선택 해제 (Deselect)

![[Pasted image 20230307094241.png]]

## 흑색 이미지에 색상 입히기
- `Image > Adjustments > Hue/Saturation` | `Ctrl + U` : 이미지에 대해 Hue, Saturation, Lightness 조절
- ^ `Colorize` 옵션을 켜주면 이미지의 Hue/Saturation 조절 가능해짐
- `레이어 패널 > Add vector mask` : 선택한 레이어와 연결된 하나의 레이어가 생김
- `Ctrl + Delete` : 레이어 배경색을 검은색으로 (Transparent)
- `Alt + Delete` : 레이어 배경색을 하얀색으로
- `Shft + KeyPad.Enter` : 상단 메뉴에 Opacity 입력칸으로 이동
