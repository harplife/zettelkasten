## Scale
- `Ctrl + Y` : 솔리드 레이어 생성
- `Ctrl + Shft + Y` : 솔리드 레이어 설정
- `Key Frame` : 정보 값을 가진 프레임
- 레이어 선택 > Ellipse Tool 더블 클릭 : 원 모형으로 마스크 생성
- `S` : Layer 패널에서 Scale 옵션이 나오게 함
- 0 프레임 > 레이어 선택 > Scale(0%)
- 15 프레임 > 레이어 선택 > Scale(0%)
- 30 프레임 > 레이어 선택 > Scale(50%)
- 45 프레임 > 레이어 선택 > Scale(50%)
