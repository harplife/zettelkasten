### Level Blueprint
- Event BeginPlay : 게임이 실행될 때 (이벤트)를 뜻함
- Print String : 문자를 출력하는 노드
	- 플레이를 하면 단순히 화면 상단 좌측 코너와 로그에 문자를 출력함
- Keyboard (키워드) : 키보드 입력과 관련된 노드 목록이 검색됨
- Event Tick : 각 프레임마다 이벤트 발생
- 변수 (Variable) : 컴파일을 우선해야 값을 넣을 수 있음
