# Rendering

## Definition
- [Rendering (Image Synthesis) - Computer Graphics | wiki](<https://en.wikipedia.org/wiki/Rendering_(computer_graphics)>) : The process of generating a photorealistic (or [[non_photo_realistic_rendering|non-photorealistic]]) image from a 2D or 3D model by means of a computer program.
	- The resulting image is referred to as the Render.
