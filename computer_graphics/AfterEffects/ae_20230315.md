# 강의 내용
- `파일명.aep` : After Effect Project, 기본 에펙 파일 포맷
- `Composition` : 새로운 프로젝트를 컴포지션이라고 함
- 29.97 FPS 사용하는 이유 : 오디오와 싱크하기 위해서
- Half-HD (1K) = `1280x720`
- Full-HD (2K) = `1920x1080`
- 황금비율 (Golden Ratio) = `16:9` : 세상에서 가장 아름다운 비율
- FHD 는 황금비율임
- `Composition Setting` | `Ctrl + K` : 콤포지션 세팅 변경 (상단메뉴 Composition 에도 있음)
- 에펙에서는 `Select` 메뉴가 없음. `Select` 은 이제 기본이 되어 생략됨.
- 영화, 게임 등 "4K" 의 의미(크기)가 다름
- `Composition Settings :: Opt(Drop Frame)` : 29.97 FPS 와 같이 소수점 FPS 는 프레임 하나를 버려주는 옵션이 있음
- `D;HH;MM;FR` : 에펙 시간단위
- 가이드를 치다?
	- [After Effects Scripting Guide | ae-scripting.docsforadobe.dev](https://ae-scripting.docsforadobe.dev/introduction/overview.html)
	- [After Effects Scriping Guide (PDF) | Adobe](https://buildmedia.readthedocs.org/media/pdf/after-effects-scripting-guide/latest/after-effects-scripting-guide.pdf)
	- [After Effects User Guide | Adobe](https://helpx.adobe.com/after-effects/using/scripts.html)
- `Duration(500)` : 영상 길이를 5초로 입력
- `Solid Settings` | `Ctrl + Y` : 솔리드 (레이어와 같은 개념) 설정
- `Solid Settings :: Opt(Make Comp Size)` : 컴포지션 사이즈와 동일하게 맞춤
- `L.Clck(Solid) > Ctrl + D` : Make Duplicate
- `Ctrl + Shft + Y` : 색 변경
- `- or +` : 타임라인 줌인/줌아웃
- `Timestamp`
	- ![[Pasted image 20230315113549.png]]
	- `L.Clck(Timestamp)` : 특정 시간으로 입력해서 점프할 수 있음
	- `Ctrl + L.clck(Timestamp)` : 시간 단위 / 프레임 단위로 변경
- 프레임 개수는 프레임 `0` 부터 셈 (5초 == 150장, 마지막 장 149)
	- "n 프레임부터" : n번 프레임 시작지점
	- "n 프레임까지" : n번 프레임 종료지점
- `[ or ]` : 타임라인에서 레이어를 Indicator 기준 시작/종료 지점으로 옮겨줌
- `Alt + [ or ]` : 타임라인에서 레이어를 Indicator 기준 시작/종료 지점으로 잘라줌
- 앞뒤로 1~2초 정도 줘야 변화를 인식할 수 있음??
- `Timeline Indicator + B or N` : Indicator 기준 재생 시작/종료 지점 지정
- Timeline Indicator + D.Clck : 재생 시작/종료 지점 전체로 지정
- Timeline + R.Clck > Trim : 재생 영역 기준 컴포지션 뒷 부분 제거
	- 시간이 반올림 된다는 것 주의!!
- `PgUp or PgDn` : Indicator 위치 조절 (프레임 1개 단위)
	- `Shft` 추가 시 프레임 10개 단위
- `Space` : 실시간 재생
	- 주의 : 렌더 플레이로 한번 해줘야 재생이 더 잘 됨
- 화면을 볼 때 파랑색 테두리는 현재 그 영역을 보고 있다는 뜻. 주의해서 살펴봐야 함!
- 파일 탐색 패널
	- ![[Pasted image 20230315122839.png]]
- `Del` : 타임라인 레이어 선택해서 제거
	- 타임라인 레이어를 제거해도 Solid 레이어는 같이 제거되지 않음
- `Alt + Scroll` : 마우스 기준 줌인/줌아웃
- 가이드를 만드는 이유 : 협업!
- `Ctrl + '` : 그리드 활성화
- `Ctrl + R` : 룰러 활성화
- `Alt + 0~4` : 채널 토글
- `Ctrl + Shft + D` : Timeline Indicator 기준 짤라서 뒷부분을 새 레이어로 분리
- `Keyframe Assistant :: Sequence Layer` : 선택한 레이어들을 순서대로 정렬해줌
	- 레이어 선택한 순서에 따라 정렬됨

## 기본 환경 설정
- `Window :: Workspace :: Default` : 기본 작업 환경
- `Edit :: Preferences :: Auto-Save :: Save every > 20 minutes` : 자동 저장 주기
- `Edit :: Preferences :: Auto-Save :: Maximum Project Versions > 5` : 이전 버전 저장 개수
