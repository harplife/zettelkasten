How to Graph Polynomial Rational Functions

Given $\frac{P(x)}{Q(x)}$ where both $P(x)$ and $Q(x)$ are polynomials and $Q(x) \neq 0$,

this guide covers how to graph the rational function using sympy and matplotlib.

In[1]:
import sympy as sp
import numpy as np
import matplotlib.pyplot as plt
In[2]:
x = sp.Symbol("x")

numer_expr = x**2 + 1
denom_expr = x+1

rational_expr = numer_expr/denom_expr

rational_expr
Out[2]:
$\displaystyle \frac{x^{2} + 1}{x + 1}$
In[3]:
f_numeric = sp.lambdify(x, rational_expr, modules=["numpy"]) # can use "math" for modules
In[4]:
'''
When dealing with vertical/slant asymptotes where zero division occurs,
it's important to define "domains" in which the function is properly defined.
Otherwise, division-by-zero will occur and it will ruin the graph.
'''

x1 = np.linspace(-10,-1.01, 400)
x2 = np.linspace(-0.99, 10, 400)

y1 = f_numeric(x1)
y2 = f_numeric(x2)

plt.plot(x1, y1, label=r"$f(x)=\frac{x^2+1}{x+1}$", color='blue')
plt.plot(x2, y2, color='blue')
plt.title("Plot of $f(x)=\\frac{x^2+1}{x+1}$")
plt.xlim(-10,10)
plt.ylim(-10,10)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.grid(True)
plt.minorticks_on()
plt.axhline(0, color='black', linewidth=.5)
plt.axvline(0, color='black', linewidth=.5)
plt.legend()
plt.show()
In[5]:
'''
If matplotlib is being difficult,
it may be easier just to use sympy to plot the graph.
'''

f = (x**2 + 1) / (x + 1)

# Plot on separate intervals
sp.plot(
    (f, (x, -10, -1.01)),   # left side of the asymptote
    (f, (x, -0.99, 10)),    # right side of the asymptote
    ylim=(-10, 10),          # optional, limit y-axis
    title='f(x) = (x^2 + 1)/(x + 1)',
    show=True
)
