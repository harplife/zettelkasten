# Balancing Chemical Equations

## Using Algebraic Method

Given this skeleton equation :

$$
CH_{4} + O_{2} \to CO_{2} + H_{2}O
$$

The equation can be balanced by following these steps :
1. Assign a variable (e.g. $a,b,c,d$) for each coefficient
2. For each element, write an equation using the variables
3. Let one variable equal to $1$
4. Solve for other variables
5. Plug the values for each variable
6. Confirm if the equation is balanced

<center>. . .</center>

First, we assign a variable for each coefficient.

$$
aCH_{4} + bO_{2} \to cCO_{2} + dH_{2}O
$$

For each element, we write an equation using the variables.

$$
\begin{align}
  C &: a = c\\
  H &: 4a = 2d\\
  O &: 2b = 2c + d
\end{align}
$$

Let a variable equal to $1$. In this case, we let $a=1$.

Solve for other variables :
- $c=1$
- $d = 2$
- $b = 2$

> [!important] If there are fractions, multiply all of the values by the least common multiple of the denominators.

We plug the values of the variables into the chemical equation :

$$
1CH_{4} + 2O_{2} \to 1CO_{2} + 2H_{2}O
$$

We can confirm that the equation is balanced by counting the elements on each side :

$$
\begin{align}
  C : 1 - 1\\
  H : 4 - 4\\
  O : 4 - 4
\end{align}
$$
