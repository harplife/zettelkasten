# Nucleosynthesis Periodic Table

This is a periodic table that shows the origin(s) of each element.
- We can see that all of Hydrogen comes from the Big Bang. Most of Helium comes from the Big Bang as well, but some part of it comes from dying low-mass stars and exploding massive stars.
